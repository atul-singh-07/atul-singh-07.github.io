What I’m Interested In

I have a strong interest in electronics and embedded systems, but I don’t see hardware as separate from software. For me, electronics becomes meaningful when it interacts with code — controlling motion, sensing the environment, and responding in real time.

How I Like to Build

I prefer project-based learning over purely theoretical study. I enjoy working on small systems that combine microcontrollers, sensors, actuators, and C/C++ code.

While building, I focus on understanding behavior, debugging issues, and improving reliability step by step — both in hardware connections and in software logic.

Things I’ve Worked On

  • Arduino-based projects involving sensors, motors, and real-time interaction
  • Servo motor control using external power sources and proper grounding
  • Ultrasonic sensor experiments for distance measurement and scanning
  • C programming concepts explored through hands-on problem solving
  • Small system-level projects that mix timing, logic, and hardware control

Currently Exploring

Right now, I’m focused on strengthening my foundations in both electronics and programming, with the goal of building more stable and well-structured systems. I’m especially curious about how low-level code interacts directly with hardware.
